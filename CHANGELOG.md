# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-10

### Added
- Initial release
- Support for HW Group Poseidon 3268 devices
- Support for HW Group Poseidon 3266 devices
- Support for HW Group SMS Gateway devices
- Temperature sensor monitoring
- Humidity sensor monitoring
- Voltage and current sensor monitoring
- Binary sensor support (contacts, alarms)
- Switch/relay control for Poseidon outputs
- UI-based configuration flow
- Automatic device discovery
- Data update coordinator for efficient polling
- HACS compatibility

### Features
- HTTP/XML API communication
- Basic authentication support
- Configurable polling interval (default 30 seconds)
- Device information and attributes
- Multiple sensor types with proper device classes
- Error handling and connection validation
